# 代理商对接邮件模板

以下模板可直接发给代理商。

---

邮件标题：

```text
[项目对接] 支付系统接入资料与联调说明
```

邮件正文：

```text
您好，

以下是贵方接入我方支付系统所需的资料，请技术同事按文档完成开发与联调。

一、接入说明
1. 技术接入文档
2. 联调检查清单
3. 商务版接入说明
4. Java / C# 示例
5. PHP 示例
6. Postman 测试说明

二、接入方式
1. 贵方服务端调用我方下单接口，创建订单
2. 我方返回支付落地页地址 landingUrl
3. 贵方将买家跳转到 landingUrl
4. 买家先进入我方首页，再自动跳转到 Stripe 支付页
5. 支付完成后，系统会先回到我方结果页
6. 我方记录订单状态后，再自动跳回贵方 returnUrl
7. 当前正式通知方式是浏览器自动回跳，不是独立 webhook

三、贵方需要提供或确认的信息
1. 代理商编号 affiliateCode
2. 贵方回跳地址 returnUrl
3. 是否启用回跳签名校验
4. 测试环境与正式环境对接人

四、注意事项
1. returnUrl 必须提前登记白名单
2. 下单请求必须带签名
3. 支付成功后不会停留在我方成功页，而是自动跳回贵方页面
4. 建议正式环境启用回跳签名校验
5. 订单支付金额以接口中的 totalAmount 为准

五、建议联调顺序
1. 先跑通下单接口
2. 再验证 landingUrl 跳转
3. 再测试支付成功、失败、取消三种场景
4. 最后确认回跳签名与订单状态更新

如需我方协助联调，请把贵方技术联系人、测试时间和测试地址发给我们。

谢谢。
```
